# Security Policy

## Scope

This repository contains the public Stoneford starter world for
WorldLines: prompts, world data, assets, and the bundled expansion zip.

Please report security-relevant issues privately if they involve:

- malicious or unsafe starter content
- secrets accidentally committed to prompts, data, or assets
- packaging or archive issues in `stoneford-worldlines-full.zip`
- instructions that could expose user secrets or create unsafe defaults

If the issue appears to live in the WorldLines engine rather than this
starter repository, still report it with the affected path or behavior.

## Supported Versions

| Version | Supported |
|---|---|
| latest `main` / latest public tag | Yes |
| older snapshots | Best effort |

## Reporting a Vulnerability

Please do **not** file public issues for suspected vulnerabilities.

Email `info@worldlines.gg` with:

- a concise summary
- affected file, asset, archive, or release
- reproduction steps or proof of concept
- impact and suggested mitigation, if known
- whether you want public credit after disclosure

Suggested subject line:

`[SECURITY][stoneford-worldlines] short description`

## Response Expectations

- Initial acknowledgement target: within 5 business days
- Follow-up status target: within 10 business days

We may coordinate a fix privately before public disclosure.
